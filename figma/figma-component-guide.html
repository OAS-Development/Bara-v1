Figma Implementation Notes

  • Use Auto Layout for all components to ensure responsive behavior
  • Create color styles for all design tokens for easy theme updates
  • Use component variants for different states (hover, active, disabled)
  • Maintain 4px grid alignment throughout
  • Icons should be 16x16px for standard UI, 20x20px for larger elements
  • Border radius: 3px (small), 6px (medium), 8px (large)
  • Use consistent naming: ComponentName/Variant/State
